# GitHub Pages Build

This folder is the static build for Paper Reader.

Publish this folder with GitHub Pages using:

- Branch: `main`
- Folder: `/docs`

The page reads local files selected by the visitor through the browser folder picker. PDFs and notes are not uploaded. Read state, notes, and drawing annotations are stored in the visitor's browser localStorage.

Server-only features such as login, multi-user libraries, AI translation, AI Q&A, related paper search, and server-side uploads remain available in the Python enhanced version from the repository root.
